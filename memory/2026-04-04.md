## 💡 技術筆記

### 打字機效果實作要點
1. 使用 `fetch('/api/system/announcements')` 動態載入公告
2. 置頂公告優先排序 (`sort` by `is_pinned`)
3. 雙層動畫：打字 → 停留 → 刪除 → 換下一則
4. 使用 `white-space: nowrap` 避免換行影響版面

### 顏色對應邏輯
```javascript
function getLevelClass(level) {
    if (level === 'urgent') return 'typewriter-urgent';  // 紅色
    if (level === 'warning') return 'typewriter-warning'; // 黃色
    if (level === 'info') return 'typewriter-info';      // 藍色
    return 'typewriter-default';
}
```
