  - `pagination` - 分頁控制
  - `toast_container` - Toast 容器
  - `loading_overlay` - 載入中遮罩
  - `empty_state` - 空白狀態

**2. global.css 更新**
- 新增 Form Section Components 章節
- 統一表單樣式標準
- 統一按鈕樣式標準
- 統一表格樣式標準
- 統一狀態標籤樣式

**3. ui_components_demo.html - 使用範例**
- 展示所有 UI 元件的使用方式
- 提供複製貼上的程式碼範本

### 使用方式

在頁面中引用 macros.html：
```jinja2
{% extends "base.html" %}
{% import "macros.html" as ui %}

{% block content %}
{% call ui.form_section('標題') %}
    {% call ui.form_row() %}
        {{ ui.input_text('id', '標籤', required=true) }}
        {{ ui.select('id', '標籤', options) }}
    {% endcall %}
{% endcall %}
{% endblock %}
```
