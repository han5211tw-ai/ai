客戶要求修改督導紀錄表：

2. 人員表現與線上回覆合併，新增員工姓名選擇欄位

#### 2. store.html 修改

#### 3. personal.html 修改
- 新增「個人督導評分（6-11項）」卡片
- 顯示人員表現 6 項評分與總分百分比

#### 4. app.py API 新增與修改
- `/api/store/employees` - 依店別取得員工列表

### 技術問題與解決

#### 問題：員工姓名載入失敗
**原因**：Python 快取 (`__pycache__`) 導致 Gunicorn 載入舊版本 app.py，新 API 路由未生效

**解決**：
```bash
rm -rf dashboard-site/__pycache__
pkill -9 -f "gunicorn"
cd dashboard-site && python3 -m gunicorn -c gunicorn.conf.py -w 4 -b 127.0.0.1:3000 app:app
```

`supervision_scores` 資料表新增欄位：

### 評分結構
| 類別 | 項目 | 滿分 |
|------|------|------|
| 環境整潔 | 門面、店內、陳列、線材、倉庫 | 10 分 |
| 人員表現 | 出勤、儀容、服務、專業、銷售、工作態度 | 12 分 |
| **總計** | **11 項** | **22 分** |

顯示方式：總分 ÷ 22 × 100 = 百分比（例如 18/22 = 82%）
