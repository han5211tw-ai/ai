# 資料來源

## OneDrive 資料夾結構

```
ai_source/
├── roster/                 # 班表資料
│   └── 班表.xlsx
├── supervision/            # 督導評分
│   └── 督導評分表.xlsx
├── feedback/               # 五星評論
│   └── (評論統計檔案)
├── performance/            # 業績資料
│   └── (Excel 績效檔案)
├── purchase/               # 進貨資料
├── inventory/              # 庫存資料
├── sales/                  # 銷貨資料
├── customers/              # 客戶資料
└── backup/                 # 備份資料
    └── openclaw/
```

## 檔案格式

### 班表.xlsx
- **分頁**: 2月班表、3月班表、4月班表...
- **欄位結構**:
  - A欄: 日期
  - B欄起: 各門市人員排班

### 督導評分表.xlsx
- **分頁**: 豐原門市、潭子門市、大雅門市
- **評分項目** (15項，每項滿分 2分):
  1. 出勤狀況
  2. 服裝儀容
  3. 服務態度
  4. 專業知識
  5. 銷售流程
  6. 門面整潔
  7. 店內清潔
  8. 產品陳列
  9. 線材管理
  10. 倉庫整齊
  11. 回覆速度
  12. 回覆態度
  13. 問題掌握
  14. 資訊完整
  15. 後續追蹤
- **總分**: 30分滿分

### 資料同步
- OneDrive 自動同步至 `~/srv/sync/OneDrive/`
- Parser 從同步目錄讀取資料
