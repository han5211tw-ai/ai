版本：v2.4.3-FINAL

NAS 一鍵部署版 v2.4.3｜分類 / 群組統一版
=================================

這版重點
--------
1. 後台「放入哪個群組」已改成與前台「故障類別」完全同步
2. 不再使用另一套獨立的預設群組名稱，避免分類混亂
3. 後台新增的故障情境若放進既有故障類別，前台會直接加到該類別底下
4. 仍可額外新增自訂故障類別，供門市自行擴充

部署方式
--------
1. 將整個資料夾放到 NAS 網站根目錄
2. 開啟 NAS 的 Web Server / PHP
3. 確認 data/db.json 可寫入
4. 開啟：
   - 登入頁：login.php
   - 前台：index.php
   - 後台：admin.php

預設帳號
--------
admin / admin123
manager / manager123
staff / staff123

注意
----
- 如果無法寫入，狀態會顯示「NAS 唯讀」
- 你也可以直接修改 data/db.json 做備份與還原
- 內建故障類別會依目前前台資料自動同步到後台下拉選單


新增功能：後台可編輯故障情境（可修改故障類別、標題、風險等級、排查步驟、可能原因、建議測試、建議處置）。


補充：後台畫面會直接顯示版本號與目前故障情境數。如果目前故障情境數為 0，右側列表不會出現編輯按鈕；請先新增一筆故障情境後再編輯。


v2.5.2 更新：前台 AI 助理已改為使用 assets/ai-logo.png 的電腦舖 AI LOGO。

v2.5.3 更新：已將 AI LOGO 重新裁切並放大，讓右下角浮動按鈕的 LOGO 幾乎填滿圓形底座；對話視窗徽章也同步放大。


v2.5.5 更新：版面管理可即時預覽 AI 助理浮動按鈕與開啟後徽章，並可分別調整大小與置中位置。
