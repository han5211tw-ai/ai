## 重要教訓

### Git 備份限制
- Git 只能還原**已提交**的檔案
- **未提交的修改**無法還原
- 修改前應先手動備份：`cp file.html file.html.backup.日期`

### 快取問題
- 瀏覽器會快取 JavaScript 檔案
- 解決方式：在 URL 後加版本號 `?v=2`
