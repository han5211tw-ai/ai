## 教訓與注意事項

1. **CSV 匯出格式**：舊系統匯出客戶 CSV 時，手機欄位在 index 17，但 Parser 預期在 index 2。未來匯出時需確認欄位順序。

2. **PID 檔案殘留**：每天早上 06:00 自動重開機可能導致資料庫同步腳本的 PID 檔案殘留。已加強腳本邏輯，自動清理殘留檔案。

3. **Cron job 設定**：isolated session 只支援 `agentTurn` payload，不支援 `exec`。晨報 job 因此失效，已修正。

4. **ERP v2 修改需謹慎**：今天修改新系統時，移除排序功能未完全移除殘留函數，導致 JavaScript 語法錯誤。未來修改需更仔細檢查。
